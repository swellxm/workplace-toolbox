# 职场提效轻量化 Web 工具箱 - 产品规划 v2.0

## 一、产品定位

**产品名称**：职场提效工具箱
**产品定位**：面向全行业职场人的零门槛、高频实用在线工具箱
**技术栈**：Python + Streamlit
**核心原则**：一看就会用、3秒出结果、数据本地处理

---

## 二、3个核心功能详细说明

### 功能1：Excel 全能处理工具

#### 解决痛点
- 职场人每周花费 3-5 小时在重复的 Excel 操作上
- 去重、分类汇总、图表制作等技术门槛高
- 格式转换、表格拆分合并等操作繁琐易错

#### 功能列表
| 子功能 | 说明 |
|-------|------|
| 一键去重 | 快速删除重复行，保留首次出现或最后出现 |
| 分类汇总 | 按指定列分组统计求和、计数、平均值 |
| 数据可视化 | 自动生成柱状图、折线图、饼图 |
| 表格拆分 | 按列值拆分为多个Sheet或多个文件 |
| 表格合并 | 多个Excel文件纵向合并为一个 |
| 格式转换 | xlsx ↔ csv ↔ txt 互转 |

#### 用户流程
```
1. 上传 Excel 文件（支持 .xlsx .xls .csv）
2. 选择处理类型（去重/汇总/可视化/拆分/合并/转换）
3. 配置处理参数
4. 点击处理，实时预览结果
5. 一键下载处理后的文件
```

#### 核心价值
- **零门槛**：告别复杂公式，一键完成专业操作
- **高效率**：原本30分钟的操作，现在30秒
- **可视化**：数据图表自动生成，直观易懂

---

### 功能2：文件批量处理工具

#### 解决痛点
- 文件整理全靠手动，效率低下
- 重命名规则复杂，容易出错
- 格式转换需要逐个操作
- 文件分类整理耗时且易混乱

#### 功能列表
| 子功能 | 说明 |
|-------|------|
| 批量重命名 | 按前缀/后缀/序号/日期等规则批量改名 |
| 格式转换 | 图片(jpg/png)、文档(docx/pdf)、音频视频格式互转 |
| 自动分类 | 按文件类型/日期/大小自动整理到文件夹 |
| 文本替换 | 批量替换文件内容中的指定文本 |

#### 用户流程
```
1. 选择文件或文件夹
2. 选择处理类型（重命名/格式转换/分类整理/内容替换）
3. 配置处理规则（实时预览效果）
4. 确认执行
5. 完成处理，查看处理报告
```

#### 核心价值
- **所见即所得**：先预览再执行，避免改错
- **灵活组合**：多种规则可自由组合
- **安全可靠**：操作前自动备份，支持撤销

---

### 功能3：职场文案全能生成工具

#### 解决痛点
- 周报月报不知道怎么写，写出来没亮点
- 竞品分析不知道从哪些维度入手
- 商务邮件格式不规范，显得不专业
- 述职报告抓不住重点，业绩展示不突出

#### 功能列表
| 模板类型 | 说明 |
|---------|------|
| 周报/月报 | 自动生成结构化工作汇报 |
| 竞品分析 | 多维度对比分析框架 |
| 活动方案 | 完整活动策划模板 |
| 商务邮件 | 专业商务沟通模板 |
| 述职报告 | 业绩展示+反思+规划结构 |

#### 用户流程
```
1. 选择文案类型（周报/竞品分析/活动方案/商务邮件/述职报告）
2. 填写关键信息（工作内容/项目名称/数据等）
3. 选择输出风格（简洁专业/详细全面/活泼轻松）
4. 点击生成，获取完整文案
5. 支持复制或导出，支持修改参数重新生成
```

#### 核心价值
- **快速上手**：填写关键信息，3秒生成完整文案
- **专业规范**：内置各场景最佳实践模板
- **灵活自定义**：支持自定义模板，保存个人常用风格

## 四、技术实现方案

### 核心依赖

| 库 | 用途 |
|---|------|
| streamlit | Web框架 |
| pandas | 数据处理 |
| openpyxl | Excel操作 |
| python-docx | Word操作 |
| Pillow | 图片处理 |
| reportlab | PDF生成 |

### 开发优先级

| 优先级 | 模块 | 原因 |
|:---::|------|------|
| P0 | Excel处理 | 核心痛点，用户覆盖面最广 |
| P1 | 文件处理 | 日常高频需求 |
| P2 | 文案生成 | 价值高但实现较复杂 |
